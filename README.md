# Lyric Generator

Streamlit app for generating lyrics using a fine-tuned GPT-2 model.

## Model Information
- Base model: GPT-2
- Fine-tuned on: Custom lyric dataset

## Deployment Structure

```
├── model/
│   ├── config.json
│   └── model.safetensors
├── requirements.txt
└── README.md
```
